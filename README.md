# AI Bewerbungsassistent 🚀

Telegram Bot für:
- Bewerbungsschreiben
- Lebenslauf PDF
- Skills Management
- Profilverwaltung

## Funktionen

✅ Telegram Bot  
✅ SQLite Datenbank  
✅ PDF Lebenslauf  
✅ Bewerbungsgenerator  
✅ Skills & Erfahrung speichern  
✅ Adresse speichern  

## Technologien

- Python
- python-telegram-bot
- SQLite
- reportlab
- dotenv

## Start

```bash
pip install -r requirements.txt
python bot.py
